# Secure Code Scanner

A Python-based Static Application Security Testing tool that scans local source code for insecure coding patterns, exposed secrets, and common vulnerability risks.

## Purpose

This project was created to strengthen my cybersecurity portfolio and demonstrate secure coding, Python programming, static analysis, and defensive security skills.

## Features Planned

- Scan local project folders
- Detect hardcoded passwords and API keys
- Detect risky Python functions such as eval() and exec()
- Detect weak hashing such as MD5 and SHA1
- Detect possible SQL injection patterns
- Severity rating: LOW, MEDIUM, HIGH, CRITICAL
- Terminal report output
- JSON report export
- Test files with safe vulnerable examples

## Tech Stack

- Python
- Regex pattern matching
- Command-line interface
- JSON reporting

## Ethical use

This tool is designed for scanning code that I own or have permission to assess. It is intended for defensive security learning and secure development pratice.
